# Usage
<!-- usage -->
```sh-session
$ npm install -g todo
$ todo COMMAND
running command...
$ todo (-v|--version|version)
todo/0.0.0 linux-x64 node-v10.16.3
$ todo --help [COMMAND]
USAGE
  $ todo COMMAND
...
```
